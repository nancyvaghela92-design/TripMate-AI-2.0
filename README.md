# ✈️ TripMate AI 2.0

An AI-powered travel planning application built with **Python**, **Streamlit**, **SQLite**, and **Google Gemini AI**.

TripMate AI 2.0 helps users plan trips, generate AI itineraries, discover hotels, calculate travel budgets, check weather, and manage saved trips—all in one place.

---
## 🌐 Live Demo

https://tripmate-ai-20.streamlit.app/

## 🌟 Features

- 🤖 AI Travel Assistant (Google Gemini)
- 🗺️ Smart Trip Planner
- 🏨 Hotel Recommendations
- ❤️ Favorite Hotels
- 💰 Budget Calculator
- 🌤️ Live Weather Information
- 📄 Download AI Itinerary as PDF
- 📊 Export Saved Trips to Excel
- 💾 Save and Manage Trips
- 📱 Clean Streamlit User Interface
- 🗄️ SQLite Database

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/nancyvaghela92-design/TripMate-AI-2.0.git
```

Go to the project folder

```bash
cd TripMate-AI-2.0
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
